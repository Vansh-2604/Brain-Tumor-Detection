# Brain Tumor Detection Using Deep Learning

A web-based Brain Tumor Detection system built using PyTorch and Flask.

## Features

- MRI Image Upload
- Brain Tumor Classification
- Confidence Score
- MRI Preview
- Responsive Dashboard UI

## Classes

- Glioma Tumor
- Meningioma Tumor
- Pituitary Tumor
- No Tumor Detected

## Tech Stack

- Python
- PyTorch
- Flask
- Bootstrap
- HTML/CSS

## Model

EfficientNet-B0 (Transfer Learning)
